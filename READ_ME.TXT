The project was created using Microsoft Visual Studio Community 2019, Version 16.11.23. 
You can run the project without Visual Studio, or 
if you prefer, you can run it within Visual Studio.

1. To run the project without Visual Studio, 
   navigate to the release folder inside the test_code directory and 
   run test_code.exe using the Command Prompt (CMD). ex) D:\test_code\Release> .\test_code.exe press enter

2. To run the project in Visual Studio, 
   simply open the "test_code.sln" file located in the test_code folder.
